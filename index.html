June 2019

My A-Level Project

My A – Level project is a web application created to help my local scout group manage their equipment and store room. It allows them to track and book out equipment as it is used for various events and returned to the storeroom, and for registering problems with individual items for the quartermaster to fix. After finishing and submitting the project, I spoke to the group's quartermaster, who was interested in implementing it. Because of this, I have recently begun working on new features and using the knowledge I have gained at University to refactor the code. As the only project shown that has been assessed, it is my largest solo project to date, using HTML, SQL, PHP and Javascript.

December 2020

Discord Bot
"CitricAmoebot"

“CitricAmoebot” is a discord bot designed to be a fun and helpful personality, currently used in my discord server. It is capable of greeting new members to the server, answering simple questions and responding to messages. This was written in python, using the Discord API, and is hosted on Heroku.

Twitch Extension
"Quick and Easy Polls"

The “Quick and easy polls “twitch extension is intended to display a poll over the top of a twitch stream in order to get a quick answer to a question. Polls can be previously created and saved for frequent use e.g. which game mode to play in the next match, or they can create an impromptu one-time poll for a new question that has come up.
