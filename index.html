August 2022

GNPS Metadata Tool

During my placement year at the University of York, I created a tool to create .tsv metadata files for datasets to be uploaded to the Chromatography web application GNPS. This was created using Python and Tkinter to create a Graphical User Interface. This tool automatically collects the names of all the files to be used in the dataset and adds their names to the "filename" column. From here, users can add additional fields with the option of adding a default value to speed up the creation of the file. Users can also use the tool to edit metadata files or import a .yaml config file to automatically add a list of fields and values. Values can be edited straight from the table, and a .tsv file is produced when the user is finished. This has been converted into a .exe file using auto-py-to-exe. Conway's Game of Life Implementation

My implementation of Conway's Game of Life, a Zero-Player Game in which the outcome is determined by the initial configuration of the game. More details can be found here. This is written in C and includes a serial version, which uses High-Performance Computing patterns, which is then parallelised using both OpenMP and MPI in two seperate version.
