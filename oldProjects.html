My A-Level Project

My A – Level project is a web application created to help my local scout group manage their equipment and store room. It allows them to track and book out equipment as it is used for various events and returned to the storeroom, and for registering problems with individual items for the quartermaster to fix. After finishing and submitting the project, I spoke to the group's quartermaster, who was interested in implementing it. Because of this, I have recently begun working on new features and using the knowledge I have gained at University to refactor the code. As the only project shown that has been assessed, it is my largest solo project to date, using HTML, SQL, PHP and Javascript.

August 2022

GNPS Metadata Tool

During my placement year at the University of York, I created a tool to create .tsv metadata files for datasets to be uploaded to the Chromatography web application GNPS. This was created using Python and Tkinter to create a Graphical User Interface. This tool automatically collects the names of all the files to be used in the dataset and adds their names to the "filename" column. From here, users can add additional fields with the option of adding a default value to speed up the creation of the file. Users can also use the tool to edit metadata files or import a .yaml config file to automatically add a list of fields and values. Values can be edited straight from the table, and a .tsv file is produced when the user is finished. This has been converted into a .exe file using auto-py-to-exe.
